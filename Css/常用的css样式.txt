1. **颜色和背景**：
   - `color`：用于设置文本颜色。
   - `background-color`：用于设置元素的背景颜色。
   - `background-image`：用于设置元素的背景图像。
   - `background-size`：用于设置背景图像的尺寸。
   - `background-position`：用于设置背景图像的位置。

2. **字体和文本**：
   - `font-family`：用于设置字体系列。
   - `font-size`：用于设置字体大小。
   - `font-weight`：用于设置字体粗细。
   - `text-align`：用于设置文本对齐方式。
   - `text-decoration`：用于设置文本装饰，如下划线、删除线等。

3. **盒模型**：
   - `width`：用于设置元素的宽度。
   - `height`：用于设置元素的高度。
   - `margin`：用于设置元素的外边距。
   - `padding`：用于设置元素的内边距。
   - `border`：用于设置元素的边框样式。

4. **布局**：
   - `display`：用于设置元素的显示方式，如块级、内联等。
   - `position`：用于设置元素的定位方式，如相对定位、绝对定位等。
   - `float`：用于设置元素的浮动方式。
   - `clear`：用于控制元素在浮动元素旁边的显示方式。

5. **动画和过渡**：
   - `animation`：用于设置动画效果。
   - `transition`：用于设置过渡效果。

6. **响应式布局**：
   - `media queries`：用于根据设备屏幕大小应用不同的 CSS 样式。
